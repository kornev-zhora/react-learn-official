# React Learning: Tic-Tac-Toe Tutorial

This is a study project based on the official [React Documentation](https://react.dev/learn). 
Specifically, it follows the hands-on tutorial: **[Tic-Tac-Toe](https://react.dev/learn/tutorial-tic-tac-toe)**.

## 🚀 Project Overview
The goal of this project is to master the fundamental building blocks of React by building an interactive game. This tutorial covers the essential concepts required to build any React application.

### Key Concepts Learned:
* **Components:** How to split the UI into independent, reusable pieces.
* **Props:** How to pass data from parent components to child components.
* **State:** How to make components "remember" things (like which square was clicked).
* **Lifting State Up:** Coordinating multiple components by moving state to a common ancestor.
* **Immutability:** Why it is important to avoid direct data mutation in React.
* **Keys:** Understanding why React needs keys to track list items efficiently.

## 🛠️ Tech Stack
* **React**
* **JavaScript (ES6+)**
* **CSS** (for styling the board and game layout)

## 📖 How to Run This Project

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/your-username/react-learn.git](https://github.com/your-username/react-learn.git)
   ```

2. **Navigate into the directory:**
   ```bash
   cd react-learn
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Start the development server:**
   ```bash
   npm start
   ```
   The app should now be running at `http://localhost:3000`.

## 📂 Project Structure
* `App.js`: Contains the main logic for the Tic-Tac-Toe board and game state.
* `styles.css`: Basic styling to render the grid and game info.
